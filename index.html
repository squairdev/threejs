Projects

squairdev.github.io - This github website is all about exploring the three.js library, and posting updates about my coding journey. Started in late October, 2024, squairdev.github.io is a website to show all of my previous and ongoing projects using WebGL in Unity, and three.js. I hope to improve and keep this project going, as I learn more from the three.js library, and WebGL.

GloShooters - Started in September 2024, GloShooters is a WebGL 2D project. The game was based off the old game "Asteroid", and a Game Jam entry. Overall, I spent 4-8 hours working on the game. In the game, you have to move around the map and shoot enemies, which follow you as you get close to them. You have limited ammo, but enemies drop some as you shoot them. You can use spare bullets in the shop. The scoring system is finally fixed (thank god, one bug down, a lot more to go). Across the whole map, there is some secrets. Go find em out for yourself! (Visit GloShooters)

ArmedHorizons - Started in October 2024, ArmedHorizons is a multiplayer Unity WebGL game. The game features a username system, shooting, running, and all of the basic first person shooter (FPS) components. When you load the website, you will be greeted by the lobby screen, where you can select a username, then hop in. You will join a lobby of up to 6 other players, in a free for all (FFA) lobby. You can shoot the enemy to deal damage, or duck behind obstacles to plan. Visit the website at: ArmedHorizons.
